# 天天生鲜在线商城系统 python578

## 项目概述

天天生鲜是一个基于Python3.x和Django2.x框架开发的小型电商购物网站。它充分利用了Django框架内建的API，并通过django-admin提供的后台管理系统进行商品和用户信息的管理。该项目适合作为Django的实战练习。

## 技术栈

- Python 3.x
- Django 2.x
- Django自带的admin管理系统

## 功能模块

### 商品浏览

- 显示商品的图片、种类、简介、库存等信息。

### 全文检索

- 支持商品种类、名称和简介的搜索。

### 登录注册

- 提供用户的注册与登录功能。

### 用户中心

- 允许用户更新个人信息、收货地址。
- 支持商品加入购物车和订单生成。

### 商品下单

- 在支付接口和企业资质支持下完成商品下单。
- 事务处理确保下单过程中的原子性和异常处理。

### 后台管理

- 管理商品和用户信息的增、删、改。
- 支持自定义样式、日志记录和权限管理。

## 系统角色

- 前端用户：负责浏览商品、下单购买等。
- 后台管理员：管理商品信息、用户数据和系统设置。

## 运行环境

- Python3.x环境
- 支持Django2.x的Web服务器

## 部署步骤

1. 安装Python3.x和Django2.x。
2. 导入数据库结构。
3. 配置Web服务器。
4. 运行项目。
